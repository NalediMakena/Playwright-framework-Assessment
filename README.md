# Playwright-framework-Assessment
Playwright JS test framework demonstrating UI and API test automation skills

## Test targets
- UI: SauceDemo (https://www.saucedemo.com)
- API: Restful-booker(https://restful-booker.herokuapp.com)

## Implementation
## UI tests:
-login
-inventory
-cart
-checkout 
-Page Object Model
## API tests
-Authentication token creation
-Booking CRUD operations (including DELETE)
-Response validation

## Github Actions
- Added Github actions

## Installation
-git clone : https://github.com/NalediMakena/Playwright-framework-Assessment.git
-cd Playwright-framework-Assessment
-npm install
-npx playwright install


## How to run tests
-Run all tests:
 npx playwright test

-Run only UI tests:
 npx playwright test tests/saucedemoUi

-Run only API tests:
 npx playwright test tests/bookerApi
